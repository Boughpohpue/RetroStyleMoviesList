# RETRO-STYLE MOVIE LIST

Oldschool look for displaying movies collection as a shelf of VHS tapes.

Designed and coded just for fun.


## INFO

* Movies list is fetched from JSON file on page load.
* Dynamic tapes creation with randomly generated stylings.
* Filtering by movie title (original and localized),
* Scrolling with different speeds using swipe, mouse wheel and keyboard (HOME/END, PAGE UP/DOWN, ARROW UP/DOWN).
